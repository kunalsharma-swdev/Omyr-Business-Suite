## DYNAMIC HOMEPAGE CATEGORY SIDEBAR

This sidebar is one of the most important parts of the homepage.

Requirements:

- Fetch every category dynamically from Supabase.
- Never hardcode category names.
- Whenever a new category is added to the database, it should automatically appear.
- Clicking a category opens that category page.
- Highlight the active category.
- Display categories alphabetically unless a database sort field already exists.
- Include a search box at the top.
- Make the sidebar scrollable if many categories exist.
- Make it sticky while scrolling on desktop.
- On mobile convert it into a slide-out drawer opened using a "Browse Categories" button.

Design:

- White background
- Rounded corners
- Soft shadow
- Light border
- Beautiful hover animations
- Premium active states using the brand colors
